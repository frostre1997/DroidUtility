About the project

DroidUtility is designed for users who want more control over their Android device without root access. It combines terminal tools, debloating features, and a clean Material 3 interface in one compact app.

Features

A focused toolset for managing, optimizing, and customizing your Android device.

🖥️

Shell Terminal

Run shell commands through Shizuku with real-time output, exit codes, and command history.

🧹

Debloat Manager

Browse installed apps, search quickly, filter by system or user apps, and uninstall or disable with one tap.

🎨

Theme Support

Switch between Light, Dark, AMOLED, and System themes with persistent saved preferences.

🔐

Shizuku Integration

Access elevated functionality without root, with permission handling built directly into the app.

📱

Material 3 Interface

Enjoy a clean, modern, and responsive design built with Android’s Material 3 visual style.

⚡

Mobile-First Development

Designed and developed for Android-first workflows, with a lightweight and practical experience.

🧩

Xposed / LSPatch Integration

Planned for v1.1.0-beta to expand advanced customization and module-based functionality.

✅

Stable and Focused

Built to stay useful, maintainable, and easy to use while keeping core tools reliable.

Requirements

What you need before using DroidUtility.

System

  • Android 7.0+ (API 24).
  • Tested up to Android 14.
  • No root required.

Shizuku

  • Shizuku must be installed and running.
  • Permission must be granted inside the app.
  • Wireless ADB or root can be used to start it.

Recommended

  • A device with enough storage for app management.
  • A stable connection for Shizuku setup.
  • Latest release for the best compatibility.
