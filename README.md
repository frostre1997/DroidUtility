# DroidUtility

A powerful, non-root utility suite for Android designed for system optimization, debloating, and advanced command execution. Built for mobile-only development.

## Features
- **Non-Root System Control:** Powered by Shizuku for privileged shell execution.
- **Debloat Engine:** Remove or disable unwanted system apps securely using JSON-based configurations.
- **Advanced Terminal:** Integrated console for ADB and SH commands with real-time output.
- **Modular Design:** Extendable via external scripts and custom configurations.
- **Mobile-First:** Fully developed and managed on Android devices.

## Requirements
- Android 10+
- [Shizuku](https://shizuku.rikka.app/) installed and running.
- [Termux](https://termux.dev/) for extended scripting capabilities.

## How to use
1. Install Shizuku and grant it permissions via Wireless ADB.
2. Grant DroidUtility access to Shizuku.
3. Use the dashboard to select your desired tweaks or run custom commands.
